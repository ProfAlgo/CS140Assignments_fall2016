
# Appendix 1 - Learing Git

## Basic operations

新建分支：
```
    # 获取主干最新代码
    $ git checkout master
    $ git pull
    # 新建一个开发分支myfeature
    $ git checkout -b myfeature
```
提交修改：
```
$ git add --all
$ git status
$ git commit -m "Your message"
```
